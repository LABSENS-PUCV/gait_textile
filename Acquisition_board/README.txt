1.- "Case" directory contains 2 STL files that are the 3D objects printed to make the case of the acquisition board.

2.- "Code" directory contains the Arduino code loaded into an STM32 "Blue pill" development board which is part of the acquisition board.

3.- "PCB" directory includes 2 files, one of them is the layout of the PCB and the other one is the schematic diagram, both were made in Eagle.
